# CV PDF Generator

Generatore automatico di PDF da HTML usando Puppeteer.

## Setup

```bash
npm install
```

## Utilizzo

```bash
node generate-pdf.js
```

Genera automaticamente `di-puorto-umberto-15-01-2026.pdf` dal file HTML corrispondente.

## Caratteristiche

- ✅ Testo completamente selezionabile
- ✅ Rendering identico all'HTML
- ✅ Nessun problema di font su Adobe Acrobat
- ✅ Dimensione ottimizzata del file
- ✅ Formato A4 con margini corretti
